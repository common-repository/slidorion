A combination of an image slider and an accordion, the Slidorion displays beautiful images along with a variable length description.

== Description ==

Slidorion is a fantastic plugin that combines an image slider and an accordion to provide a platform that allows you to present information in a beautiful manner.

The plugin is easily customisable and gives you a variety of options such as:

* Autoplay
* Transition easing
* Large variety of effects (fading, sliding, overlapping, random)
* Choose the first slide
* Pause the player on hover
* Interval between transitions
* Speed of the transitions

You can create as many Slidorion's as you want, as they are all stored as separate entities. All you need to do to place it within any page, is call it's shortcode like this: [slidorion name="foo"]

== Installation ==

Download the

e.g.

1. Login in to WordPress and search for Slidorion
2. Click install and then activate

OR

1. Download the Slidorion plugin and unzip it.
2. Upload it to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress


Creating A Slidorion

1. Go to the Slidorion menu in the admin navigation and click "New Slidorion".
2. Give the Slidorion a name, select the options you want and upload an image for each slide, along with some text.
3. Save the Slidorion
4. Note down it's designated shortcode and go to a post you want to use it in.
5. Place the shortcode [slidorion name="YOUR NAME"], replacing YOUR NAME with the name you want to use
6. DONE!!
